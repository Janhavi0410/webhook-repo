# TechStaX Webhook Assignment

This project is a simple Flask backend that demonstrates
how to receive and handle webhook data.

## Features
- Flask API server
- Webhook endpoint (/webhook)
- Receives JSON payloads and logs them

## How to run
1. Install dependencies:
   pip install flask

2. Run the app:
   python app.py

3. Test webhook:
   Send a POST request to:
   http://127.0.0.1:5000/webhook
